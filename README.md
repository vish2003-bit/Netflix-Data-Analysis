# Netflix Data Analysis

## Overview

This project analyzes Netflix Movies and TV Shows data using Python, Pandas, and Matplotlib.

## Objectives

- Explore Netflix content distribution
- Compare Movies and TV Shows
- Identify top content-producing countries
- Analyze content growth trends

## Tools Used

- Python
- Pandas
- Matplotlib
- Google Colab

## Key Findings

- Movies dominate Netflix's content library.
- The United States produces the highest number of titles.
- Netflix experienced rapid content growth between 2016 and 2020.
